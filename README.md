HelpOn é o sistema de gestão de chamados da companhia aérea fictícia FlightOn.
A plataforma centraliza todo o ciclo de vida das solicitações dos passageiros/usuários — desde o registro até o arquivamento — abrangendo áreas críticas como bagagens,
reembolsos, assistência especial etc. O sistema conta com fluxos personalizados para três perfis distintos (Passageiro, Atendente e Administrador), garantindo permissões
e visões específicas para cada necessidade. Tecnologias utilizadas: AngularJS, HTML5, CSS3, JavaScript e a implementação do Bando de Dados no Supabase.
